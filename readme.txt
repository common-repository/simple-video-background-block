The plugin that allows to add YouTube video backgrounds.

== Description ==

This plugin adds a Gutenberg block container to your site so that you can add YouTube videos as a background to it.  YouTube video backgrounds is a single block that nests other blocks.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/video-background` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Can I use other blocks inside of this? =

Yes, YouTube Video Background is a single block that nests other blocks.
